# Lava Minigames!!

Guide:  
https://www.youtube.com/watch?v=zM87XEbS60M

Turn your world into a slowly collapsing wasteland.

Lava Minigames!! adds two powerful items that allow you to start and stop a destructive corruption event. Once activated, the world around you begins to decay as random terrain is consumed by the void and lava emerges from the ground.

* **Features**
  * **Start Item**
    * Right-click the Start Item to begin the event.
    * Every 1 second, a random vertical column is completely erased from the world.
    * Every 1.5 seconds, random lava blocks appear around the player.
    * Destruction follows the player as they move.
    * The world slowly transforms into a dangerous volcanic wasteland.

  * **Stop Item**
    * Right-click the Stop Item to immediately halt the event.

* **Designed For**
  * Challenge maps
  * Minigames
  * Survival events
  * Content creation
  * Custom SMP events
  * Speedrun challenges

* **Notes**
  * The effect is server-side.
  * Terrain destruction is permanent.
  * Use with caution on important worlds.
  * Recommended for backups, challenge worlds, and event servers.

* **Warning**
  * ⚠️ This mod permanently destroys terrain.
  * Always back up your world before using it.
